About the Application

The application is being built in response to COVID-19 complications and students need to be able to browse a course catalog to register remotely. While you could browse a document, that is not very friendly. The end result of this application will allow for CSV uploads for course catalogs from a SIS.

You can start now by going to the Schools page and browse the catalog for the fake high school. Current functionality allows for courses to be added with an online form.
